# C-Pokemon-Game
A terminal-based Pokémon RPG built in C/C++ featuring dynamic terrain generation and an OOP battle system. The Pokémon game uses the ncurses library.

```bash
# Build the executable using the Makefile
make

# Run the game
./pokemon
```
